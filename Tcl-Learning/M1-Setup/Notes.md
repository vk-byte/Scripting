# Notes
## Setup
For a beginner starting to learn Tcl, below are the steps to setup Tcl. <br>
To run Tcl locally, we need a text editor and Tcl interpreter. <br> 
- Text Editor - choose any you're comfortable with from options like Windows Notepad++, OS Edit command, Brief, Epsilon, EMACS, and vim or vi etc. <br>
- Tcl Interpreter - It can be obtained from Tcl distributions. Go to official website https://www.tcl-lang.org. Download one among BAWT, Magicsplat, IronTcl. Magicsplat is recommended for Windows, and go for version 1.18.x based on Tcl 8.6.x as latest release might not be in wide use in industry.<br>

Voila ! You can start with Tcl on Command Prompt or any text editor. Check out the next module to kick off !
