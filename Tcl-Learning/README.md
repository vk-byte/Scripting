# Tcl Learning Repository
This repository contains my Tcl Learning notes and practice programs for VLSI automation.

## Quick background
Tcl stands for Tool Command Language.
<br>Tcl was created specifically to make software tools scriptable and automatable.
<br>In VLSI, Tcl acts as the glue between engineers and EDA tools: 
<br>`Engineer --> Tcl script --> EDA Tool --> Chip Design Results`
